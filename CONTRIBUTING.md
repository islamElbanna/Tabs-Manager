# Contributing to Tabs Manager

First off, thank you for considering contributing to Tabs Manager! It's people like you that make open source such a great community.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Tabs-Manager.git
    cd Tabs-Manager
    ```
3.  **Create a branch** for your specific changes:
    ```bash
    git checkout -b feature/amazing-feature
    # or
    git checkout -b fix/annoying-bug
    ```

## Development Workflow

1.  Make your changes in your local branch.
2.  **Test your changes**:
    -   Load the extension in Chrome (`chrome://extensions` > "Load unpacked").
    -   Verify that your changes work as expected.
    -   Check the "Errors" button on the extension card in `chrome://extensions` for any runtime errors.
3.  **Code Style**:
    -   Follow the existing code style (standard JavaScript).
    -   Use 4 spaces for indentation.
    -   Ensure variable names are descriptive.

## Submitting a Pull Request

1.  **Commit your changes** with a clear message:
    ```bash
    git commit -m "Add: dragging capability for tabs"
    ```
2.  **Push to your fork**:
    ```bash
    git push origin feature/amazing-feature
    ```
3.  **Open a Pull Request** (PR) on the main repository.
    -   Provide a clear title and description of what you've done.
    -   Link to any relevant issues (e.g., "Fixes #123").

## Reporting Bugs

If you find a bug, please create a new issue on GitHub. include:
-   **Steps to reproduce**: How can we see the bug ourselves?
-   **Expected behavior**: What should have happened?
-   **Actual behavior**: What actually happened?
-   **Screenshots**: If applicable.

## Feature Requests

We love new ideas! Please open an issue to discuss your idea before implementing it to ensure it aligns with the project's goals.

## Code of Conduct

Please note that we have a Code of Conduct, and we expect all contributors to adhere to it. Be kind, respectful, and constructive.

**Happy Coding!**
